# morettie_inc
1. Power BI Project: Moretti Sales Analysis
An interactive sales analysis created to assist stakeholders in tracking important indicators, seeing patterns, and making data-driven choices is demonstrated in this Power BI project. Sales performance, product trends, regional comparisons, and client segmentation are all visualized on the dashboard.

2. Key Features:
   * Dynamic visualizations & KPIs
   * Sales trends: monthly, quarterly, yearly
   * Top products & brands analysis
   * Sales Channel segmentation & insights
   * Drill-through & cross-filtering functionality
3. Tools & Technologies
   *Power BI Desktop – for report creation
   *DAX (Data Analysis Expressions) – for custom calculations and measures
   *Power Query (M language) – for data cleaning and transformation
   *Power BI Service for publishing dashboards
   * MS Excel for dataset input
  
4. 📌 Business Goal
To measure monthly sales performance across product lines and identify areas of under‑performance for strategic focus.
